# Iris Ridge Regression Project

## Project Overview

This project uses the Iris dataset to evaluate different regression models and model selection workflows. The project includes simple linear regression, multiple linear regression, polynomial features, ridge regression, and grid search to find the best model parameters.

## Workflows 
[Open Workflows](https://github.com/yunachou05/data-science-learning-notes/blob/main/iris-ridge-regression-analysis%20practice/data/Iris%20dataset_model%20development%20design%20.png)

## Objectives

- Load and explore the Iris dataset
- Create summary tables by species
- Build simple linear regression models
- Build multiple linear regression models
- Apply polynomial ridge regression
- Use GridSearchCV to find the best alpha and polynomial degree
- Export model results and plots

## Methods Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Ridge Regression
- Grid Search
- Cross-validation
